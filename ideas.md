# Ideas de Design — Legacy Guild (Free Fire)

## Opção A — Cyberpunk Neon Warfare
<response>
<text>
**Design Movement:** Cyberpunk / Dark Tactical
**Core Principles:**
- Contraste extremo entre preto profundo e azul neon elétrico
- Tipografia agressiva e angular para transmitir poder e dominância
- Elementos de interface que remetem a HUDs militares e telas de jogo
- Sensação de urgência e adrenalina em cada seção

**Color Philosophy:**
- Fundo: #000000 / #050a14 (preto absoluto e azul-noturno)
- Neon primário: #00d4ff (azul ciano elétrico)
- Neon secundário: #0066ff (azul royal intenso)
- Accent: #00ffcc (turquesa neon para destaques)
- Texto: #e0f7ff (branco frio com leve toque azulado)

**Layout Paradigm:**
- Assimétrico com diagonais cortando seções
- Painéis sobrepostos com efeito glassmorphism escuro
- Linhas de scan e grid de fundo sutil

**Signature Elements:**
- Bordas neon com glow/box-shadow azul
- Texto com efeito de glitch em títulos principais
- Linhas diagonais cortando seções (clip-path)

**Interaction Philosophy:**
- Hover com intensificação do glow neon
- Botões com efeito de "energia carregando"
- Scroll com parallax suave

**Animation:**
- Entrada de elementos com slide + fade
- Pulso de glow em elementos de destaque
- Partículas de luz flutuando no hero

**Typography System:**
- Display: Rajdhani Bold / Orbitron (futurista, angular)
- Body: Exo 2 (legível, tecnológico)
- Accent: Rajdhani Semibold para subtítulos
</text>

## Opção C — Neon Abyss (ESCOLHIDA)
<response>
<text>
**Design Movement:** Neon Abyss / Cyberpunk Gaming
**Core Principles:**
- Abismo negro como tela, neon azul como luz que rasga a escuridão
- Assimetria intencional que cria tensão visual e dinamismo
- Efeitos de luz volumétrica e glow que dão profundidade ao preto
- Identidade visual de guilda de elite — exclusividade e poder

**Color Philosophy:**
- Fundo base: oklch(0.05 0.02 240) — preto com leve matiz azul
- Neon primário: oklch(0.75 0.25 220) — azul ciano elétrico
- Neon secundário: oklch(0.55 0.28 250) — azul royal intenso
- Accent quente: oklch(0.80 0.20 200) — azul-turquesa para CTAs
- Texto principal: oklch(0.95 0.03 220) — branco frio
- Texto secundário: oklch(0.65 0.08 220) — cinza-azulado

**Layout Paradigm:**
- Hero com layout diagonal: texto à esquerda, imagem à direita cortada em ângulo
- Seções alternadas com clip-path diagonal para criar fluxo dinâmico
- Cards com bordas neon e glassmorphism escuro
- Navegação horizontal com indicadores neon

**Signature Elements:**
- Box-shadow neon em cards e botões (0 0 20px rgba(0,212,255,0.5))
- Linhas diagonais de corte entre seções
- Texto com text-shadow neon em títulos principais

**Interaction Philosophy:**
- Hover intensifica o glow neon progressivamente
- Botões com animação de "carregamento de energia"
- Links com underline neon animado

**Animation:**
- Hero: partículas de luz flutuantes com framer-motion
- Seções: slide-up com fade ao entrar no viewport
- Títulos: efeito glitch sutil no hover
- Cards: elevação com intensificação do glow no hover

**Typography System:**
- Display: Orbitron (futurista, angular) — para títulos e logo
- Body: Exo 2 (legível, tecnológico) — para texto corrido
- Mono: Share Tech Mono — para estatísticas e dados
</text>
<probability>0.09</probability>
</response>

## DECISÃO FINAL: Opção C — Neon Abyss
Design escolhido por melhor representar a identidade de uma guilda de elite em Free Fire,
combinando o abismo do preto com a energia do azul neon para criar uma experiência visual
única, poderosa e memorável.
